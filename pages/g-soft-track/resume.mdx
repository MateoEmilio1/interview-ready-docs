# Preparando el Resume

El 90%+ de las applications a un puesto laboral se filtran por el currículum. Errores básicos de formato y contenido reducen tus chances de conseguir entrevistas
y reducen tus chances de dar una buena impresión durante todo el proceso de entrevistas.

En este video vamos a armar el currículum de Victor Vigón en base a su perfil de LinkedIn y siguiendo la Checklist de currículums.

Para el final de la sesión vas a tener un resume de base para presentar en empresas, más el conocimiento para armar currículums para oportunidades especiales.
